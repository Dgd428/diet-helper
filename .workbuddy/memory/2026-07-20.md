# 2026-07-20 工作日志

## 项目：饮食助手（健康减肥软件）

### 产品讨论与需求确认
- **产品定位**：面向减肥人群+家庭成员的饮食规划工具，降低每日"吃什么"的决策成本
- **核心功能**：
  - 按时段自动切换首页显示（早餐6:00-10:30 / 午餐10:30-15:00 / 晚餐15:00-20:00 / 禁食20:00-6:00）
  - 多成员支持：减脂餐 + 营养餐并列显示
  - 锁定机制：锁定某餐后其余自动平衡热量，减脂餐7天内不重复
  - 周末：允许仪式感大菜 + 一顿光明正大的放纵餐
  - 采购清单：周五18:00弹出气泡，自动合并重复食材
  - 标记已吃：可选打卡，温柔不催促，随时可补标记
  - 食材库：搜索食材→保存方法+选购技巧+关联菜品
- **产品性格**：温暖、温柔，不爹味
- **名称**：饮食助手
- **技术路线**：先做纯HTML单页 → 验证 → 后续考虑小程序

### 已完成
- 创建 `data/recipes.json`：30道菜品（减脂21道 + 营养9道），每道菜含完整步骤（火候+时间）
- 创建 `data/ingredients.json`：45种食材的保存方法、选购技巧、关联菜品
- 创建 `preview.html`：菜谱和食材库预览页面，支持搜索和Tab切换

### 正式网站构建（23:04-）
- 创建 `index.html`：完整单页应用
- 核心功能已实现：
